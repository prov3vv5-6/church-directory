# Next Steps

## Set Up Supabase Database

1. Go to supabase.com and sign up for a free account
2. Click "New project"
3. Give it a name like `church-directory`
4. Set a database password — save this somewhere safe, you'll need it
5. Pick the region closest to you (probably East US)
6. Click "Create new project" and wait ~2 minutes for it to provision

Once done, we'll grab the connection string and wire it into the backend.
